comments for run backend :
npm init
npm i express

npm i nodemon {in package.json ->scripts-> add start }
npm i dotenv {for env file}

//for connect with mongoDB
npm i mongoose


express → Node.js web framework

mongoose → MongoDB ODM (database connection)

cors → Allow frontend to access backend from a different port

dotenv → Load environment variables from .env

nodemon → Automatically restart server on file changes (dev only)
